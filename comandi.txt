Eseguire questi comandi nella cartella server:
Per connetterti alla macchina aws tramite ssh:
ssh -F ssh_config_server.dev sin-post-quantum-test-ec2-server

Installare i paccchetti necessari al server. 
ansible-playbook -i inventoryserver.ini install_env.yml

Copia del progetto nella macchina AWS:
ansible-playbook -i inventoryserver.ini deploy_server.yml

Eseguire questi comandi nella cartella client:
Per connetterti alla macchina aws tramite ssh:
ssh -F ssh_config_client.dev sin-post-quantum-test-ec2-client

Installare i paccchetti necessari al client:
ansible-playbook -i inventoryclient.ini install_env.yml

Copia del progetto nella macchina AWS:
ansible-playbook -i inventoryclient.ini deploy_client.yml

Per mostrare le risorse utilizzate nella macchina:
top; htop; ps aux | grep apt

Rimozione cartella con permessi di root:
sudo rm -rf WebAppPostQuantumServer

Per copiare la cartella report da AWS sul host:
scp -i AWSPROVA.pem -r ubuntu@13.51.175.236:/home/ubuntu/WebAppPostQuantumServer/report ./report
